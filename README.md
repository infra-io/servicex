**Servicex** 是一些服务生态的集合包，开箱即用。

[Read me in English](./README.en.md)

### 📝 功能特性

* 集成常用的 grpc 拦截器
* 集成 trace id 生成和传递
* 随机字符串生成，随机令牌串生成
* 栈信息获取，自动设置 max proc 数值
* 带字符串解析的 Duration 类型
* 快速获取时间的 Clock 类型

_历史版本的特性请查看 [HISTORY.md](./HISTORY.md)。未来版本的新特性和计划请查看 [FUTURE.md](./FUTURE.md)。_

### 🔧 使用方式

> $ go get -u github.com/FishGoddess/servicex
